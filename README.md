# MobiCasa

MobiCasa is an interactive, modern furniture website focused on **home furniture and interior products**. The site includes sections such as Home, Products, About, and Contact, with a responsive and visually appealing design.

---

## Project Structure

- `index.html` – Main HTML file of the website.
- `style.css` – CSS file for styling and layout.
- `script.js` – JavaScript file for interactive features, like the hamburger menu.
- `/asstes/` – Folder containing all images used in product cards and the About section.

---

## Key Features

1. **Header and Navbar:**
   - Logo and navigation links.
   - Hamburger menu for mobile devices.
2. **Home Section:**
   - Engaging headline and slogan.
   - “Shop Now” button.
   - Responsive background image.
3. **Product Categories:**
   - Product cards with `category-title`, `product-name`, and images (`src`).
   - “Shop Now” button for selected products.
   - Responsive grid layout that adjusts for mobile, tablet, and desktop.
4. **About Section:**
   - Information about MobiCasa.
   - Image and descriptive text in a clean layout.
5. **Contact Section:**
   - Contact form to submit messages.
   - Contact info and office location.
